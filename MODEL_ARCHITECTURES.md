# MINDSIGHT Model Architectures

| Domain | Name | Algorithm | Model Type | Artifact | Input | Output |
|--------|------|-----------|-----------|----------|-------|--------|
| 1 | Big Five Personality | Generalized Rating Model (GRM) IRT Scoring | Parametric IRT | `domain1_grm_parameters.pkl` | EXT1-3, EST1-3, AGR1-3, CSN1-3, OPN1-3 | 5 trait scores (-4.0 to +4.0) |
| 2 | Rosenberg Self-Esteem | Deterministic Sum + Reverse-Coding + Age/Gender Percentile Lookup | None (Deterministic) | `domain2_self_esteem_percentiles.json` | age, gender, Q1-10 (or RSE1-10) | RSE Score 0-40 + Classification + Age/Gender-Adjusted Percentile |
| 3 | Mood & Sleep | PHQ-9 Deterministic + K-Means Clustering | Unsupervised (Optional) | `domain3_mood_sleep.pkl` (optional) | DPQ010-090, SLQ300, SLQ310 | PHQ-9 Sum, Sleep Duration, Clinical Phenotype (optional) |
| 4 | Digital & Social | Tree-Based Regression + SHAP Explainer | XGBoost / LightGBM | `domain4_digital_social.pkl` | IAT1-10, loneliness1-6, age, gender | Internet Addiction Sum, Loneliness Score, Depression Risk |
| 5 | Occupational Burnout | Ordinal Cumulative Classification (tier) + Quantile Regression (index) | XGBoost, monotonic-constrained | `domain5_ordinal_ge_*.json` (x3), `domain5_quantile_q*.json` (x3) | age, gender, work_hours_per_week, meetings_per_day, work_life_balance_score, job_satisfaction_score, deadline_pressure_score, autonomy_score, stress_score, social_support_score, + 3 engineered features | P[tier >= Moderate/High/Severe], Burnout Index (10th/50th/90th percentile) |
| 6 | Severe Clinical Screening | Multinomial Logistic Regression (Symptom Cluster Screener) + Isolation Forest (Atypical Presentation Detector) | Logistic Regression + Isolation Forest | `domain6_clinical.pkl` | unwanted_thoughts, repetitive_behaviors, overthinking, mind_going_blank, avoidance_social_activity, panic, hypervigilance, sleep_disturbances, low_energy (9 binary features) | Symptom Cluster (8 classes) + Atypical Presentation Flag |

## Key Notes

- **Domain 1**: True IRT theta estimation over grid `[-4.0, +4.0]` (81 points)
- **Domain 2**: Deterministic RSE sum + reverse-coding, unchanged; age and gender (previously collected but unused) now feed an empirical percentile lookup so results can be reported relative to peers, not just an absolute 0-40 cutoff. No IRT/ML component -- deliberately kept deterministic given the instrument's validated scoring rule.
- **Domain 3**: PHQ-9 always computed deterministically (validated instrument, scoring rule not to be replaced by ML); optional K-Means phenotyping if model exists
- **Domain 4**: Fallback to raw sum if model missing; SHAP explains tree-based predictions
- **Domain 5**: Redesigned from a single XGBoost regressor + post-hoc tier binning to two complementary heads: 3 cumulative ordinal classifiers (P[tier >= Moderate/High/Severe], correctly treating tiers as ordered rather than nominal) and 3 quantile regressors (10th/50th/90th percentile) giving the Burnout Index a genuine uncertainty interval instead of a bare point estimate. Monotonic constraints enforced on the 8 work-related + 3 engineered features (signed direction known a priori); no constraint on age/gender (direction not assumed). SHAP TreeExplainer per head. **Training data (`tech_burnout_2026_clean.csv`) is a synthetic Kaggle dataset, not real occupational survey data.** `stress_score` correlates with the `burnout_score` label at r=0.854 (R^2=0.73) and dominates SHAP attribution accordingly -- this reflects the synthetic label's construction, not a validated real-world "stress is the primary driver of burnout" finding, and should not be presented as clinical/occupational-psychology insight without independent validation on real data.
- **Domain 6**: Redesigned after investigation showed the real `ocd_symptoms_clean.csv` (8,304 rows, confirmed genuine) has no severity label at all -- the original "Condition Code (0-3)" severity-screening design was not achievable with this data. The raw `Disease` column (22 distinct values) was found to have only 8 statistically distinguishable symptom signatures; the other 14 diseases + "No illness" are indistinguishable from baseline (noise-level prevalence on all 9 features) and are collapsed into `No_Detectable_Signal`. `Anxiety` and `Bipolar Disorder` were confirmed non-separable on this feature set (both elevate only `sleep_disturbances`, at nearly identical rates, with no other differentiating feature) and are merged into `Sleep_Disruption_Nonspecific`. Model 1 (multinomial logistic regression) predicts the most likely of these 8 clusters directly from `Disease`-derived ground truth, with per-class coefficients stored transparently in metadata. Model 2 (Isolation Forest) is a separate, complementary signal -- not severity or combination-rarity, but atypical/multi-cluster presentation detection, validated post-hoc by confirming flagged rows show meaningfully higher symptom counts than non-flagged rows (2.4 vs. 0.38 in the current training run). Full mapping and rationale in `domain6_clinical_metadata.json`'s `cluster_map` and `design_notes`.
